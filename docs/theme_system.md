# Theme System

## Palette

- Deep green: primary civic action and brand trust
- White: surfaces and content balance
- Black: typography and strong accents
- Red: destructive actions only
- Grey: helper text and secondary UI

## Flutter Files

- `lib/core/constants/app_colors.dart`
- `lib/core/constants/app_assets.dart`
- `lib/core/theme/app_theme.dart`

The non-3D splash and icon are used first. The 3D assets are registered for later use.
