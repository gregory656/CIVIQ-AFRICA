# Moderation Rules

## Phase 1 Baseline

- Destructive actions use red only.
- Reports and appeals are stored separately from profile data.
- Moderation actions must include action, target, moderator when available, reason, and timestamp.
- Appeals remain available from the global drawer/support area.

## Later Work

- Automated suspicious behavior detection.
- Report review queues.
- Clear user-facing dispute and account recovery flows.
